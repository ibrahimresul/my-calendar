# 📅 MyCalendar - Cloud-Based Personalized Agenda Web Application

MyCalendar is a modern, responsive, and feature-rich personalized agenda web application designed with an elegant glassmorphic UI. It offers dynamic theme switching, event management with category color codes, and seamless data synchronization using both LocalStorage and Firebase Firestore.

---

## 🚀 Features

* ⏳ **Real-Time Sync**: Fully integrated with Firebase Firestore for live cloud operations.
* 💾 **Offline Capability**: Smart fallback mechanism to LocalStorage when database connection is unavailable.
* 🎨 **Glassmorphism UI**: High-end modern look with 4 different themes (Ocean, Forest, Sunset, Night).
* 📅 **FullCalendar Integration**: Monthly, weekly, and daily interactive view modes with smooth mouse scroll navigation.
* 🛡️ **Admin Dashboard**: Comprehensive control panel to monitor total records, export all events as JSON backup, or wipe local/cloud data safely.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Custom properties/variables for themes, Flexbox/Grid)
* **Libraries:** FullCalendar.js v5, FontAwesome v6
* **Backend/Database:** Firebase Firestore (Compat SDK)
